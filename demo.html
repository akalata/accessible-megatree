Menu Navigation Tips

The following menu has 3 levels. Use left and right arrow keys to navigate between menus and submenus. Use up and down arrow keys to explore within a submenu. Use enter to activate. Within a submenu, use escape to move to top level menu parent. From top level menus, use escape to exit the menu.
